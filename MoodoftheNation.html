Mood of the Nation - IBM

In this project IBM tasked my group and I to design a web-application that makes analysis of data from twitter easy and provide quick data visualisations.

Problem Statement

’Many social media analytics tools are designed for professional marketers. Therefore, they are not particularly consumable to a typical social media user who wants to understand how their social media message/request has been received’

As stated an easy to use interface with sophisticated visualisation was desired. During the project I developed my skills in HTML, CSS and JavaScript to create a visually appealing and intuitive front-end user interface. I also gained experience in working with various APIs such as the Twitter Developer API and IBM's proprietary machine learning tools.

Solution

The solution required various APIs such as the Twitter Developer API and the Firestore Database API which were interfaced and connected together in NodeRED. NodeRED is a browser based flow editor, built on node.js, designed for wiring together various APIs and online services which can then be used to create JavaScript functions. The final product features a dynamic map with coloured indicators designating how many tweets, retweets and sentiment. Also featured
