Vector Network Analyser

This project was my final coursework for my Instrumentation Module.

This project involved heavy use of LTSpice and MATLAB. LTSpice was used to simulate the system and create/collect the data for the circuits. The data was then imported into MATLAB using a custom script. The functions of the Vector Network Analyser were then simulated in a MATLAB script.

Below are lower level designs outlining the specific designs of building blocks I used for the overall system. the first image shows a dual input differential amplifer setup used to amplify voltage differences across the current sense resistors. The middle block is a 2nd order low pass filter and the right block is a mixer used to form the basis of the In-Phase and Quadrature modulation blocks.
